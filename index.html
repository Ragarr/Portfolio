Encriptador de Áreas de Imágenes

Proyecto de Criptografía

Esta aplicación permite encriptar secciones específicas de imágenes en el lado del cliente. Luego, almacena las imágenes parcialmente encriptadas en un servidor.

La arquitectura de la aplicación utiliza varios mecanismos de criptografía para proteger la información:

  • Almacenamiento de contraseñas con hashes usando Scrypt y salt aleatorio.
  • Cifrado de imágenes con AES-CTR y claves derivadas de contraseñas mediante PBKDF2-HMAC.
  • Verificación de integridad de imágenes con HMAC-SHA256.
  • Uso de certificados digitales para firmar imágenes en el cliente y verificar las firmas en el servidor.

Con una implementación completa, usando autoridades reales y separación física de componentes, el sistema sería seguro.
